# map-marker

## Deskripsi

Map Marker adalah aplikasi web sederhana untuk menampilkan lokasi-lokasi penting di Banda Aceh menggunakan Leaflet.js. Aplikasi ini menampilkan peta interaktif, marker pada setiap lokasi, pop-up informasi, dan daftar lokasi di sisi kiri agar pengguna lebih mudah memilih titik yang ingin dilihat.

## Apa yang Telah Saya Lakukan

- Menyusun peta interaktif berbasis Leaflet.js.
- Menambahkan marker untuk beberapa lokasi di Banda Aceh.
- Membuat pop-up informasi untuk tiap marker.
- Menampilkan daftar lokasi di sidebar kiri.
- Menambahkan aksi klik pada daftar lokasi untuk mengarah ke titik marker di peta.
- Membuat tampilan responsif agar tetap nyaman dipakai di layar kecil.
